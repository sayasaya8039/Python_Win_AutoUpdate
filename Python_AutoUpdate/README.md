# Python AutoUpdate v1.0.0

Windows用Python自動アップデートツール

## 使い方

1. `Python_AutoUpdate.exe` を起動
2. 自動的にバージョンチェックが実行されます
3. 新しいバージョンがある場合、「アップデート」ボタンが有効になります
4. 「アップデート」をクリックしてダウンロード・インストール

## 機能

- Python公式サイトから最新バージョン情報を自動取得
- 現在のインストール済みバージョンと比較
- ワンクリックでダウンロード・インストール（管理者権限で実行）

## 注意事項

- インターネット接続が必要です
- インストール時には管理者権限が必要です
