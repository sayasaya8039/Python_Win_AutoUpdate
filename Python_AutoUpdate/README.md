# Python AutoUpdate v1.1.0

Windows用Python自動アップデートツール

## 使い方

1. `Python_AutoUpdate.exe` を起動
2. 自動的にバージョンチェックが実行されます
3. 新しいバージョンがある場合、「アップデート」ボタンが有効になります
4. 「アップデート」をクリックしてダウンロード・インストール

## 機能

- Python公式サイトから最新バージョン情報を自動取得
- 現在のインストール済みバージョンと比較
- ワンクリックでダウンロード・インストール（管理者権限で実行）
- **定時自動チェック** - 毎日指定時刻に自動でバージョンチェック
- **自動インストール** - 新バージョンを自動でインストール
- **システムトレイ常駐** - バックグラウンドで動作

## オプション設定

「オプション設定」ボタンから以下が設定可能:

- 定時チェックのON/OFF・時刻設定
- 自動インストールのON/OFF
- トレイに最小化
- Windows起動時に自動起動

## 注意事項

- インターネット接続が必要です
- インストール時には管理者権限が必要です
- 設定は `%APPDATA%\PythonAutoUpdate\settings.json` に保存されます
